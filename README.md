# Астронавигатор

Публичный сайт-сервис на базе ведической астрологии (джйотиш) — аналитический
подход без мистики. Сделан по двум ТЗ (`TZ_sayt_astronavigator` и
`TZ_navigator_dnya`).

## Статус

**Этап 1 — фронт на заглушках.** Все страницы и потоки собраны; расчётный
слой (`compute/`) возвращает синтетические данные в правильной структуре.
Реальный движок (Swiss Ephemeris / pyswisseph, айянамша Лахири) подключается
следующим этапом переиспользованием движка Telegram-бота — контракт API при
этом не меняется.

## Архитектура (3 слоя, ТЗ раздел 1)

```
compute/           СЛОЙ 1 — расчёт (сейчас заглушки; дальше pyswisseph)
  natal.py           натальный каскад (Махадаша, Антардаша)
  daily.py           дневной слой (Луна, Чандра-бала, Тара-бала, оценка)
templates/         СЛОЙ 2 — презентация (Jinja + static/)
static/
content/templates.yaml  СЛОЙ 3 — тексты интерпретации по ключу из слоя 1
config/scoring.yaml     коэффициенты оценки (правка без деплоя)
app.py             Flask: маршруты страниц + /api/calculate + /api/daily
```

## Страницы

| Путь | Что |
|---|---|
| `/` | Главная: манифест, архетипы, вход в расчёт |
| `/navigator` | Навигатор дня: каскад + «Сегодня» + гейт-полоса 7 дней |
| `/method` | О методе |
| `/order` | Заказ разбора |
| `/privacy`, `/offer` | Юридические (заглушки) |
| `POST /api/calculate` | Натальный расчёт (JSON) |
| `GET /api/daily` | Дневной слой + 7 дней (JSON) |

## Запуск локально

```bash
pip install -r requirements.txt
python app.py
# http://localhost:5000
```
